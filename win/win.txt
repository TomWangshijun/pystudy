Todo: Modified in Windows 10, python 2.7

第一步，我们先来安装Python
鼠标右键我的电脑  -> 属性 -> 点击高级系统设置 -> 点击环境变量 -> 点击PATH -> 在最后面加上我们的Python安装路径 -> 点击确定

注意的问题是，添加PATH路径的时候，要在最后面添加一个分号。

第二步，我们安装pip。
下载地址是：https://pypi.python.org/pypi/pip#downloads
下载 tar.gz包。解压后进入
python setup.py install
添加环境变量，在PATH最后添加：
C:\PythonXX\Scripts
